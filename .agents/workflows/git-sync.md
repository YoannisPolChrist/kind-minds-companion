description: Vollautomatischer Git Sync (Add, Commit, Push) mit generierter Commit-Nachricht
---

// turbo-all

Führe diese Schritte der Reihe nach aus:

1. Führe den Befehl `git status` aus, um zu überprüfen, welche Dateien geändert wurden.
2. Analysiere die Änderungen und erstelle eine passende, prägnante Commit-Nachricht im Format: `type(scope): message`.
3. Führe den Befehl `git add .` aus, um alle Änderungen zu stagen.
4. Führe den Befehl `git commit -m "<Commit-Nachricht>"` aus, wobei `<Commit-Nachricht>` durch deine generierte Nachricht ersetzt wird.
5. Führe den Befehl `git push` aus, um die Änderungen an das Remote-Repository zu senden.
